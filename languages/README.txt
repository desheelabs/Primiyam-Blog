# Primiyam Blog - Translation Instructions

## How to Translate This Theme

This theme is translation-ready. To translate it into your language:

### Method 1: Using Loco Translate Plugin (Recommended)
1. Install "Loco Translate" plugin from WordPress
2. Go to Loco Translate > Themes
3. Select "Primiyam Blog"
4. Click "New Language"
5. Choose your language
6. Start translating

### Method 2: Using Poedit
1. Download Poedit from https://poedit.net/
2. Open Poedit
3. Create new translation from POT file
4. Select language
5. Translate strings
6. Save as .po and .mo files in /languages/ folder

### File Naming Convention:
- primiyam-blog-hi_IN.po (Hindi)
- primiyam-blog-ar.po (Arabic)
- primiyam-blog-es_ES.po (Spanish)
- primiyam-blog-fr_FR.po (French)

### Text Domain:
primiyam-blog

## Translatable Strings Include:
- Menu items
- Widget titles
- Button text
- Search placeholder
- Comments form
- Footer text
- Meta information
- Error messages
