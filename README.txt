== Description ==

Primiyam Blog is a modern and responsive WordPress theme designed specifically for bloggers and content creators. It features a clean, professional design that puts your content first while providing powerful customization options.

= Key Features =

* Fully Responsive Design
* Custom Logo Support
* Custom Header & Background
* Multiple Widget Areas (Sidebar + 3 Footer Areas)
* Social Media Integration
* Color Customization
* Translation Ready
* SEO Optimized
* Fast Loading
* Cross-browser Compatible
* Mobile-Friendly Navigation
* Reading Progress Bar
* Back to Top Button
* Smooth Scrolling
* Lazy Loading Images
* Custom Comment Form
* Post Navigation
* Breadcrumb Support
* Clean Code
* Well Documented

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= How do I change the theme colors? =

Go to Appearance > Customize > Theme Colors. You can customize primary and secondary colors.

= How do I add a logo? =

Go to Appearance > Customize > Site Identity and upload your logo.

= How do I set up the menu? =

Go to Appearance > Menus, create a new menu, and assign it to the "Primary Menu" location.

= Does this theme support widgets? =

Yes, the theme has 4 widget areas: 1 sidebar and 3 footer widget areas.

= Is the theme translation ready? =

Yes, the theme is fully translation ready with .pot file included.
